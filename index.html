Name: ccna-study-tracker · Set to Public · Check "Add a README file"

Repository access: Only select repositories → pick ccna-study-tracker
Permissions: Contents → Read and Write → Generate
Step 3 — Enable GitHub Pages: repo → Settings → Pages
Source: Deploy from a branch · Branch: main · folder: / (root)
